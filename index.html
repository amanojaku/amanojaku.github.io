About me

I am a developer, guitarist and fan of acoustic finger style blues. In late 2014 I got a new guitar, and started to learn some really old blues songs that I couldn't find any music for. I wanted to have a disciplined approach to music and playing guitar, so I dedicated myself to learning a new song each week for the next year

What's it all about

Combining my passions, I created a Word Press theme and launched a site to hold all the songs I was learning. Hopefully my site be used as a resource for other finger style guitarists.

I've created this GIT project to detail the challenges I've faced and the learning and knowledge I've gathered in creating this project.

This site was created using the Bootstrap Jumbotron template and Font Awesome icons
